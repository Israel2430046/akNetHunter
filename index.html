About Kali NetHunter

Kali NetHunter is a free and open-source mobile penetration testing platform for Android devices, based on Kali Linux. It provides a robust platform for security professionals and enthusiasts to conduct advanced penetration testing and security assessments on the go.

Features

  • Wireless 802.11 frame injection
  • USB HID keyboard attacks
  • BadUSB MITM attacks
  • DuckHunter HID Attacks
  • Supports multiple Android devices
